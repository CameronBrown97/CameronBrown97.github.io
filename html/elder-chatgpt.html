Elder-ChatGPT

The purpose of the website to allow people to take a cognitive impairment test for free. They will have
their voice recordings processed by an AI model to determine if they need to go see a doctor.

My contribution to the back end was focused on the admin dashboard. The following image shows what the
dashboard looks like when first directed to it. On the left side of the screen, there is a list of all
the users that took the exams (test data).

Now there can be many people who take the exam. That is why I had created a way of filtering out users
Based on their first name. This will more than likely change to filter based on the users' email
addresses.

When you select the one you want it will have two other filters for you to choose. The first one is which
of the two tests you would like to see. Once that filter is selected, you'll be able to select which day
they took the test. Once the date is selected, it will then display the answers that the user entered.

The idea of this is to be able to edit the users' answers if they had accidentally entered the 'wrong'
answer.

The live version of the website can be found here: saskatchewan.elderchatgpt.com
